El presente repositorio contiene los archivos solicitados para realizar la evaluación
del estudiante Othon Gabriel Suarez Mendez en el modulo 3 del curso Herramientas y --
Destrezas del Big Data Ambiental. Se acordó durante clase que la evaluación consistiría 
en crear un pipeline simple con la paqueteria Drake en Rstudio.

Mi pipeline consiste en una serie de funciones que tienen como objetivo importar un -
archivo de Excel, el cual contiene un conjunto de datos correspondientes a los resul-
tados en forma de score de la interacción de mas de 12 mil compuestos obtenidos de la 
base de datos ZINC con el stio d1 de la proteina VCP/p97 usando el programa Autodock-
Vina.

El archivo se importa, se transforma a zscore y luego se escribe un xlsx con los zscore
de los resultados.

Saludos.
